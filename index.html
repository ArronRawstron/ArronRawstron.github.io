C++ Command Line Paper Games

Features (See README.txt on GitHub for more details)

This is my first actual game project where it is a collection of basic games that can be played at the command line.
These games can either be played with another person on the same PC or with against the PC depending on which game you play.
Local Score Tracking: Saves scores of win, loss and draw on device with usernames
BattleShips: Simple battleships against an AI

Connect 4: Connect 4 against a second user
Hangman: Hangman where the computer picks a work and you try to guess

This Website

This is the website mentioned which has been made to show what I have worked on, what my qualifications are and what experience I have.

Python Shape Area Calculator

This was a basic piece of code which I made when first being taught the python language.
The code allows users to select a shape, (triangle, rectangle or circle), then allows the user to enter the 1 or 2 numbers needed for the calculation.
It then allows the user to keep entering what they think the answer is until they get it correct or give up.
The program also has a basic GUI window with the help information in to experiment with tkinter.

Java Glass Cutting

This assignment was to code a solution to a version of the bin packing problem.
The code had to include both the First Fit and Next Fit algorithms.
It was also required that the generator class was modified and completed.
The assignment also specified that both correctness and performance testing had to be coded fully.
It was also required that sorting was implemented to determine whether sorting made a difference to the performance of the algorithms.

Java Basic Sorting Algorithms

The aim of this assignment was just to implement multiple types of sorting algorithms including insertion sort, quicksort and an undefined sort given to us.
It was also required that the comparisons required for the sorts were counted to evaluate the performance.

Java Telephone Directory

This was an assignment where the aim was to create a directory in Java to store and manage entries in a telephone directory.
It was also required to give 3 different implementations using different data structures to compare the performance.
It also had to have some form of GUI to manage the directory.

Java Sandwich Builder

This assignment was to create a sandwich making program that allowed the user to create a sandwich from ingredients and also total up the cost.
It was required that the user could create their own sandwich or select a premade one from the menu.
The program also had to display the receipt detailing what ingredients were included.

Java Tax Calculator (No Longer Runs)

This was one of the first proper Java programs I had written and the main aim was to make a program that would calculate tax given an income.
It was also required that it created a bar chart of tax values when given a set of data.
